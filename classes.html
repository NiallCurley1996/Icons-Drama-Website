Classes & Information

There are drama and singing classes everyday between Monday-Friday. the class times for
the different age groups are shown below:


6-8 yrs:

Singing classes on Monday are between 18:00 - 20:00


9-10 yrs:

Tuesday drama class is between 13:00 - 15:00
Tuesday singing classes are between 18:00 - 20:00


11-12 yrs:

Wednesday drama classes are between 13:00 - 15:00.
Singing classes on Wednesday are between 19:00 - 20:30


13-14 yrs:

Thursday drama classes are from 12:00 - 14:30.
Thursday singing classes are between 17:00 - 19:00


15-16 yrs:

Friday drama class is from 12:30 - 14:30.
Singing classes on Friday are between 18:00 - 20:00
